# Headphones Landing Page

This project is a responsive landing page for a fictional headphones company, designed based on the Figma template provided by ALX.

## Features
- Mobile responsive layout (switches at 480px)
- Hover states for buttons and links
- Max-width set to 1000px centered
- Clean and modern UI following the design spec

## Technologies
- HTML5  
- CSS3 (No frameworks)

## Instructions
Open `index.html` in your browser to view the project.
